# The Unfinished Game

A personalized chess experience with AI-generated poetic messages for every move.

## Features

- Complete chess rules (castling, en passant, pawn promotion, check/checkmate/stalemate)
- AI-generated poetic messages powered by Groq's LLaMA 3.3 70B
- Personalized
- Beautiful dark theme with smooth animations
- Production-ready serverless backend

## Local Development

```bash
npm install
npm run dev
```
